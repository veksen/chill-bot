# Chill-Bot

> WIP of a custom bot written in TypeScript on Discord.js

## Install

```bash
npm i
# or
yarn
```

## Usage

Copy the `.env.example` to `.env` and set the variables

```
TOKEN=yourbottoken
ATLAS_USERNAME=username
ATLAS_PASSWORD=yoursecretpassword
```

Run the bot

```bash
npm run dev
# or
yarn dev
```
